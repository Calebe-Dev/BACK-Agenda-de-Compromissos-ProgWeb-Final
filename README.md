# Agenda de compromissos (Back-End agenda de compromissos - prog-web Projeto Final.)

### 1 - acesse : https://start.spring.io/

### 2 - Selecione as opções:
Projeto: Maven
Linguagem: Java
Springboot: 3.3.5

### 3 - Sete os nome conforme seu projeto

### 4 - Selecione a versão de java equivalente a instalada em sua maquina ou anterior e o empacotamento como Jar

### 5 - Adicione as dependencias:

* H2 Database SQL Provides a fast in-memory database that supports JDBC API and R2DBC access, with a small (2mb) footprint. Supports embedded and server modes as well as a browser based console application  
* Spring Boot DevTools DEVELOPER TOOLS Provides fast application restarts, LiveReload, and configurations for enhanced development experience.  
* Spring Web WEB Build web, including RESTful, applications using Spring MVC. Uses Apache Tomcat as the default embedded container.  
* Spring Data JPA SQL Persist data in SQL stores with Java Persistence API using Spring Data and Hibernate.  Clique em Gerar
